# Exercise - job offers

Exercise - Backend for job offer posting service

## Using & developing the app

###### Installation

```bash
$ npm install
```

###### Running
```bash
# development
$ npm run start

# production mode
$ npm run start:prod
```

###### Testing
```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
